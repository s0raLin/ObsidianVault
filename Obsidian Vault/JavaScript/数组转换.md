## map方法
想象一下，将一个数组的每个元素的值都乘以 3，同时又不想改变原数组
```js
const numbers = [5, 10, 15, 20, 25]; //原数组


let numbersByThree = new Array(numbers.length)
numbersByThree[0] = numbers[0] * 3;
numbersByThree[1] = numbers[1] * 3;
// ...
```
这种方式繁琐且低效,这时候就用到map()

map() 的语法：

```js
array.map(function(currentValue, index, arr));
```
参数说明：

- **function（必需）**：对数组中每个元素执行的函数
- **currentValue（必需）**：当前正在处理的元素
- **index（可选）**：当前元素的索引（从 0 开始）
- **arr（可选）**：原始数组本身

### 使用 map()

```js
// 先定义这个函数
function multiplyByThree(element) {
  return element * 3;
}

// 把这个函数作为参数传给 `map()`：
let numbersByThree = numbers.map(multiplyByThree);

console.log(numbersByThree); 
// [15, 30, 45, 60, 75]
```

### 使用匿名函数
```js
let numbersByThree = numbers.map(function (element) {
  return element * 3;
});
```
更现代、更常见的写法是使用 **箭头函数**：
```js
let numbersByThree = numbers.map((element) => {
  return element * 3;
});
```
### 在 map() 中使用内置函数
```js
const numbers = [5, 10, 15, 20, 25];

console.log(numbers.map(Math.sqrt));
// [2.23, 3.16, 3.87, 4.47, 5]
```
`Math.sqrt`是JavaScript的内置函数，用于求平方根,你也可以尝试 `Math.floor` 或 `Math.ceil`，观察不同的效果

## join方法
`join()` **只能用于数组**。将一组单词用同样的分隔符分开，并组成一句话.(它**不会修改原数组**，并返回一个**字符串**)
`join()` 接收一个**字符串分隔符**作为参数，例如：
- `" "`（空格）
- `","`
- `"-"`

```js
const words = ["JavaScript", "is", "truly", "interesting"];

// 要看到结果，需要把它保存到一个变量中
const phrase = words.join(" ");
console.log(phrase);
// JavaScript is truly interesting
```

## toString方法
`toString()` 顾名思义，就是把数组中的元素转换为字符串
它**不需要任何参数**。
```js
const numbers = [1, 2, 3, 4, 5];
console.log(numbers.toString());
// 1,2,3,4,5
```
`toString()` 不仅能用于数组，还可以用于：
- Number
- Function
- Object
- 等其他对象类型
