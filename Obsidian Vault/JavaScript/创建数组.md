## 使用字面量
```js
// 字面量写法
let arr = [element_1, element_2, element_3, element_n];

let arr1 = ['JetBrains', 'Hyperskill'];
let arr2 = [45, 34, 23];
let arr3 = ['JavaScript', 12];
```

## 使用new和Array()构造函数

当你只向 `Array()` 传入**一个数字参数**时，它表示数组的**长度**，会创建一个包含指定数量**空槽位**的数组：
### 传入单个参数
```js
// 单个参数
let arrayCreate = new Array(6);

console.log(arrayCreate.length); // 6
console.log(arrayCreate[0]); // undefined
```

### 传入多个参数
如果你向 `Array()` 构造函数传入**多个参数**，这些参数会作为数组的元素：
```js
// 多个参数
let platforms = new Array('JetBrains', 'Hyperskill', 'WebStorm', 'PyCharm');

console.log(platforms.length); // 4
console.log(platforms[0]); // JetBrains
```
## 创建/修改数组的常用方法
### `fill()`

`fill()` 方法会**修改原数组**，将数组中的元素替换为一个固定的值
该方法最多接收三个参数：

```js
fill(value, start, end)
```
参数说明：
- `value`：用于填充数组的值（必需）
- `start`：开始填充的位置（可选，默认是 `0`）
- `end`：结束位置（不包含该索引）（可选，默认是数组长度）

示例:
```js
const myNumbers = [1, 4, 7, 10, 15];

// 从索引 1 开始，到索引 3 结束（不包含 3），填充为 0
console.log(myNumbers.fill(0, 1, 3)); // [1, 0, 0, 10, 15]

console.log(myNumbers.fill(13)); // [13, 13, 13, 13, 13]

// 从索引 2 开始填充为 5
console.log(myNumbers.fill(5, 2)); // [13, 13, 5, 5, 5]
```

### 结合 `new Array()` 和 `fill()`
```js
let numberArray = new Array(6).fill(2); // [2, 2, 2, 2, 2, 2]
```
> 注意,在使用 `fill()` 方法之前，**一定要先声明并初始化数组**。

### `from()`
`Array.from()` 方法可以从**类数组对象**或**可迭代对象**（例如 `Map`）创建一个新的数组
语法如下:
```js
Array.from(object, mapFunction, thisValue)
```
参数说明：
- `object`：要转换成数组的对象（必需）
- `mapFunction`：对每个元素执行的映射函数（可选）
- `thisValue`：执行 `mapFunction` 时 `this` 的值（可选）

#### 示例一：使用字符串
```js
Array.from('Hello'); // ['H', 'e', 'l', 'l', 'o']
```
#### 示例二：使用函数参数

```js
function createArray(...arguments) {
  return Array.from(arguments);
}

createArray(2, 4, 6); // [2, 4, 6]
```

#### 示例三：传入映射函数
```js
Array.from([3, 5, 7], x => x * x); // [9, 25, 49]
```
### `of()`
它会根据传入的参数创建一个新数组
语法
```js
Array.of(element_1, element_2, ..., element_n)
```
示例：
```js
Array.of(15); // [15]

Array.of(101, 202, 303); // [101, 202, 303]

Array.of(undefined); // [undefined]
```

### `Array.of()` 与 `Array()` 的区别

当参数是**整数**时，两者的行为不同：
- `Array.of(10)` → 创建一个包含单个元素 `10` 的数组
- `Array(10)` → 创建一个长度为 10、元素为 `undefined` 的数组
