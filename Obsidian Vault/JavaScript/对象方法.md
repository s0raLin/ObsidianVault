## 对象方法创建

我们来尝试理解对象方法是如何创建的。为此，请看以下示例：

```javascript
let product = {
  name: "Microwave",
  description: "With oven mode",
  price: 398
};

product.giveDiscount = function() {
  console.log("You have a 10% discount!");
};

product.giveDiscount(); // You have a 10% discount!
```

在这段代码中，我们通过编写方法允许对象报告产品的折扣`giveDiscount`。

## 语法简写

还有一种更简洁的方法可以将函数设置为属性。例如：

```javascript
let person = {
  greetings: function() {
    console.log("Hello");
  }
};
```

我们也可以省略关键词`function`：

```javascript
let person = {
  greetings() {
    console.log("Hello");
  }
};
```

简化的语法不仅可以节省时间，还可以提高代码的可读性。

## _“this”_ 关键字

在自然语言中，我们可以使用代词来指代特定的人、动物或物体。在 JavaScript 中，你也可以使用关键字`this`来指代特定的对象：

```javascript
let user = {
  firstName: "Elliot",
  lastName: "Alderson",
  fullName() {
    return this.firstName + " " + this.lastName;
  }
}; 

console.log(user.fullName()); // Elliot Alderson
```

在上面的例子中，借助`this`，该`fullName`函数访问了存储在`user`对象中的`firstName`和`lastName`字段，因为它在对象内部声明了 。

对嵌套对象应用关键字`this`可能会造成一些混淆。在这种情况下，记住，关键字`this`指的是它被使用的方法所在的对象。

## _“this”_ 和箭头函数

箭头函数 _不能_ 绑定`this`。换句话说，箭头函数没有自己的绑定`this`。

如果在箭头函数内部使用`this`，它的值将取自以通常方式声明的外部函数：

```javascript
let movie = {
  name: "The Thirteenth Floor",
  year: 1999,
  getInfo() {
    let arrow = () => console.log("The movie " + this.name + " was shot in " + this.year);
    arrow();
  }
}; 

movie.getInfo(); // The movie The Thirteenth Floor was shot in 1999
```

事实上，JavaScript 中的具体用法`this`并不局限于上述示例。这里只介绍了使用这个关键字的基础知识，还有很多内容值得学习。
