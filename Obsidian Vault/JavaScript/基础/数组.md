数组用于在一个变量中保存多个元素。
数组是一种特殊的对象，使用`typeof`返回的结果是`object`

## 声明数组

### `let marks = new Array()`
### `let marks = []`

## 初始化数组
### 方法一(推荐)

```js
// 声明并初始化数组
let marksOne = [45, 50, 44, 30, 29]; // 仅包含整数

let marksTwo = ["forty-five", "fifty", 44, 30, "twenty-nine"]; // 字符串和整数混合
```
### 方法二

```js
let marksOne = new Array(45, 50, 44, 30, 29); 
// 创建一个包含 5 个元素的数组

let marksTwo = new Array(5); 
// 创建一个长度为 5 的数组，所有元素都是 undefined

let marksThree = new Array("45"); 
// 创建一个只包含字符串 "45" 的数组
```

### 方法三
#### 逐个初始化元素
```js
let marks = new Array(5);

marks[0] = "forty-five";
marks[1] = "fifty";
marks[2] = 44;
marks[3] = 30;
marks[4] = "twenty-nine";
```

## 索引位置(Index)

**索引位置**来访问或赋值数组元素的方式。
- **数组索引从 0 开始**
- 访问超出数组范围的索引会返回 `undefined`
- 访问数组中尚未初始化的位置，也会返回 `undefined`

```js
let marks = [45, 50, 44, 30, 29];
marks[10] = 65;
console.log(marks);
```


## 数组长度(length)
`length` 属性用于返回数组中元素的数量：
```js
const languages = ["C", "C++", "Python", "Java", "JavaScript"];

let len = languages.length; // 5
```

## isArray()
判断一个对象是否为数组
语法:
```js
Array.isArray(obj)
```
示例:
```js
let marks = [45, 50, 44, 30, 29];
Array.isArray(marks); // true

let marksOne = "45";
Array.isArray(marksOne); // false
```
