对象是一种非原始数据类型，用来表示一组**无序的属性集合**

每个属性由 **键（key）和值（value）** 组成，它们之间用冒号 `:` 分隔。

- 键只能是 **字符串（string）** 或 **Symbol**（一种数据类型，后续会介绍）
- 值可以是 **任意数据类型**

## 创建对象
### 空对象
你可以创建一个没有任何属性的空对象

```js
let book1 = {}; 
let book2 = new Object();

console.log(typeof book1); // object 
console.log(typeof book2); // object
```

### 多个属性的对象

```js
let person = {
	name: "张三",
	age: 23
}
```

## 引用属性

### 访问属性

```js
console.log(person.name); // "张三"
```

### 设置/添加属性

```js
person.love = "篮球"
```

### 删除属性

```js
delete person.love
```
