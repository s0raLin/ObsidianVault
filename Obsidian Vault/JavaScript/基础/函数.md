## 函数声明

要声明一个函数，你需要：

1. 关键词`function`
2. 必需的标识符应反映其功能。函数标识符可以包含字母、数字、下划线和美元符号（规则与变量相同）。通常，它们是动词
3. 一对圆括号。圆括号内可以包含用逗号分隔的参数名：`(parameter1, parameter2)`。参数是传递给函数的输入，以名称的形式存在，其行为类似于局部变量
4. 函数体由一对花括号组成

```javascript
function sayHi(name) {
  alert('Hello' + ' ' + name);
}

sayHi("Joe"); // "Hello Joe"
```

声明的函数不会立即执行。它们是为后续使用而准备的，将在被调用时执行。

函数声明的主要特点 是，JavaScript 解释器会在执行其余代码之前先定义函数声明。这种函数行为称为函数**提升（hoisting**）。JavaScript 的默认行为就是函数提升，即将函数声明移到代码顶部。因此，我们可以在定义函数之前就调用它，这段代码也能正常运行

```javascript
sum(2, 3); // 5

function sum(a, b) {
  return a + b;
}
```

## 函数表达式

**函数表达式**定义了一个函数，该函数是表达式（通常是赋值表达式）的一部分。

```javascript
let sayHi = function(name) {
  alert('Hello' + ' ' + name);
}
```

函数表达式通常定义为将匿名函数赋值给一个变量。

这样我们就可以创建一个没有函数标识符的匿名函数，这是函数表达式和函数声明之间的主要区别。

与函数声明不同，函数表达式必须在调用之前定义。函数表达式是在创建它们的表达式执行期间创建的。

在这种情况下，函数将在赋值操作期间创建。

```javascript
let sum = function(a, b) {
  return a + b;
}

sum(2, 3);
```

## 箭头函数表达式

**ECMAScript**是 JavaScript 标准的一个修订版本，有不同的版本（ES6、ES21 等）。ES6 于 2015 年发布，是 JavaScript 发展史上的一个里程碑。它引入了**箭头函数**。

在现代标准 JavaScript 中，可以使用**箭头函数**来声明函数表达式。箭头 函数的语法比普通函数表达式更简洁。

```javascript
let sum = (a, b) => {
  return a + b;
}

sum(2, 3);
```

## 函数构造函数

也可以使用名为`Function()`的内置 JavaScript 函数构造器来定义函数

```javascript
const mult = new Function("a", "b", "return a * b");

let x = mult(4, 3); // x = 12
```

实际上，这种函数定义方式很少用到。上面的例子等同于：

```javascript
const mult = function (a, b) {return a * b};

let x = mult(4, 3); // x = 12
```
