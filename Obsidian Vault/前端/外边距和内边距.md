## 外边距(Margin)
`margin`CSS属性设置HTML元素之间的空隙
![[Margin.png]]
### Margin使用场景
#### 居中元素
如果元素有固定宽度，可以使用 `margin: auto` 将其水平居中
> 注意：`margin: auto` 只有在块级元素宽度已设置时才有效。

#### 突出元素
当你希望某个元素单独显示，不被其他元素靠近时，可以使用 margin 来实现间距。

## 内边距(Padding)
`padding`CSS属性用于在HTML元素内部创建空间，位于元素边框之内
![[Padding.png]]

### Padding使用场景
#### 增加元素尺寸
padding会增加元素尺寸以容纳内边距
#### 背景覆盖内边距区域
如果元素有背景颜色，padding区域也会显示背景
#### 内容周围留出空隙
padding可以在元素内部的内容与边框之间创建间距,使内容更易读

## Margin和Padding的多值语法
### 单值语法
单个值用于四个方向
```css
element {
  margin: 10px;  /* 四个方向都是 10px */
  padding: 10px; /* 四个方向都是 10px */
}
```
### 双值语法

第一个值应用于上下，第二个值应用于左右：

```css
element {
  margin: 10px 20px;  /* top&bottom: 10px, left&right: 20px */
  padding: 10px 20px; /* top&bottom: 10px, left&right: 20px */
}
```

### 三值语法

第一个值应用于上，第二个值应用于左右，第三个值应用于下：

```css
element {
  margin: 10px 20px 30px;  /* top:10px, left&right:20px, bottom:30px */
  padding: 10px 20px 30px; /* top:10px, left&right:20px, bottom:30px */
}
```

### 四值语法

依次为上、右、下、左，按顺时针方向设置：

```css
element {
  margin: 10px 20px 30px 40px;  /* top right bottom left */
  padding: 10px 20px 30px 40px; /* top right bottom left */
}
```

> 小贴士：如果需要，可以使用负值设置 margin（例如 `margin-left: -5%`），但 padding 不允许负值。
