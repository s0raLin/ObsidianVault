HTML文件中，描述越靠前的元素，浏览器渲染时就越早显示。
默认情况下，页面元素的`position`属性值对应`static`,改变这个属性就称为更改定位

## 绝对定位(absolute)
* 如果没有父元素，那么元素会相对于浏览器视口进行定位，也就是浏览器视口充当父元素
![[无父元素的absolute.png]]
* 如果有父元素且父元素的 `position` 不是 `static`，元素会相对于父元素边界定位
![[有父元素absolute.png]]

语法如下：

```css
element {
  position: absolute;
}
```

### 例子
四个蓝色方块采用的是**普通定位(position属性默认是sattic的)**
![[普通定位.png]]改变元素2的position属性为absolute
![[absolute定位.png]]
绝对定位的本质上将元素从正常流中抽离出来，达到了绝对定位位于静态定位平面上的效果
![[绝对定位抽离.png]]

大多数情况下，绝对定位元素的宽度和高度会自动调整以匹配其内容。如果你不手动设置元素大小（即将其设置为auto），它们也可以拉伸到其全部宽度或高度。要实现这一点，您需要将top和bottom或left和right属性设置为 0。

#### 填充高度
```css
.parent {
  position: relative;
  border: 1px solid blue;
  height: 150px;
  width: 200px;
}
 
.child {
 position: absolute;
 width: 50px;
 background-color: rgb(250, 135, 135);
 top: 0;
 bottom: 0;
}
```
![[填充高度.png]]

#### 填充宽度
填充宽度的方法类似

```css
.child {
 position: absolute;
 height: 50px;
 background-color: rgb(250, 135, 135);
 left: 0;
 right: 0;
}
```
![[填充宽度.png]]

## 相对定位(relative)
允许你不将元素从文档流中移除的情况下，移动位置，其他元素仍保持其原始位置。语法如下
```css
element {
  position: relative;
}
```
### 例子
这里有四个相对位置的方块
![[四个位置相对的方块.png]]
现在将偶数编号的方块向右移动50px,奇数编号的方块向左移动50px
![[奇数向左，偶数向右.png]]

**相对定位(relative)** 和 **绝对定位(absolute)** 不同的一点是相对定位不会从文档流中移出元素。这种行为允许你将元素移动到任何位置，但不破坏布局，因为这份空间仍然会被占据

![[相对定位和绝对定位的区别.png]]


## 固定位置
固定位置类似于绝对定位，但元素是相对于浏览器视口进行定位的。即使页面滚动，元素的位置保持不变。语法如下
```css
element {
  position: fixed;
}
```

### 示例
```html
<div class="body">
     <header class="header">
       <h2>Images are from Pixabay</h2>
     </header>
     <div class="content">
       <img src="https://cdn.pixabay.com/photo/2015/03/26/09/52/chain-link-690503_1280.jpg">
       
       <img src="https://cdn.pixabay.com/photo/2020/02/05/18/38/jungle-4822003_1280.jpg">
       
       <img src="https://media.istockphoto.com/photos/tropical-forest-trees-in-sunlight-and-rain-picture-id178502011?s=612x612">
       
       <img src="https://media.istockphoto.com/photos/celeste-river-costa-rica-picture-id1222643210?s=612x612">
       
     </div>
</div>
```
```css
h2 {
  margin: 0;
  text-align: center;
  position: fixed;
  background-color: lightblue;
  width: 100vw;
  height: 100px;
}

img {

  display: block;
  margin-left: auto;
  padding-bottom: 10px;
  margin-right: auto;
  width: 70%;
}
```

## 粘性定位
从功能上看，它介于固定定位和相对定位之间：元素在页面滚动到一定位置之前是相对定位的，滚动到该位置之后，元素的位置将固定。

```css
element {
  position: sticky;
}
```

### 示例

```html
<div class="container">
  <div class="main blue"></div>
  <div class="main purple"></div>
  <div class="main red"></div>
  <div class="main yellow"></div>
  <div class="main orange"></div>
</div>
```
```css
.container {
  margin-left: 300px;
}
.main {
  position: sticky;
  top: 10px;
  width: 120px;
  height: 120px;
  margin-bottom: 150px;
  border: 1px solid rgba(0, 0, 0, .2);
}

.blue {
  background: #13b4ff;
}

.purple {
  background: #ab3fdd;
}

.red {
  background: #ae163e;
}

.yellow {
  background: #FFFF00;
}

.orange {
   background: #FFA500;
}
```
要使这种定位方式生效，您需要指定一个限制点。这些块会在达到此限制点时停止，在本例中，该限制点是 top 属性`10px`。当滚动页面并使元素达到该限制点时，元素将像下面的动画所示那样一个叠一个地堆叠起来：
![[粘性定位.gif]]如果移除 top 属性，定位将类似于相对定位：
![[移除限制点的粘性定位.gif]]
