React是一个非常强大的JavaScript库，它允许你创建带有各自逻辑的各种 UI 元素，并在代码中重复使用

## 特性
在React中，你会把应用拆分成若干个相互独立的部分，这些部分被称为**组件(components)**。

**React能让你的应用更加高效**
DOM(文档对象模型 Document Object Model)是浏览器创建的一个对象，用来表示文档结构。当用户点击按钮并加载了新数据时，浏览器会生成新的DOM，然后刷新整个页面，即使新旧DOM之间只差了一个单词。这种策略并不高效，还会导致内存浪费。

为了解决这个问题，React使用了**组件树(React Components tree)**，这是一种轻量级的DOM表示方式，它可以只更新真正发生变化的组件。

使用React进行调试也更加容易。React在父组件和子组件之间有一套清晰的嵌套和数据流模式，是开发者能够更快定位错误，也能更好地理解整个应用的结构
