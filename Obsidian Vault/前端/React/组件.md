## 什么是组件?
组件(Components)是简单、独立、动态和可复用的代码片段
与JavaScript函数一样返回内容，不过React组件返回的是HTML元素

React中有两种组件类型:
* 函数组件(Functional Components)
* 类组件(Class Components)

### 示例: Card组件
```jsx
import React from 'react'

function Card() {
  return (
    <div className="card">
      <div className="input-container">
        <input type="text" placeholder="Email"/>
        <input type="text" placeholder="Password"/>
      </div>
      <div className="button-container">
        <button>Login</button>
        <button>Signup</button>
      </div>
    </div>
  )
}

export default Card;
```

## 创建组件

### 引入React
```js
import React from 'react'
```

### 定义App函数
```js
function App() {

}
```

### 返回HTML元素
```js
function App() {
	return <h1>Hello, world</h1>
}
```
React 在底层使用了 **Babel 编译器**，它会把 JSX 转换为浏览器能识别的 JavaScript。
因此，只要在 React 环境中运行，这个函数就是一个 React 组件

## 用箭头函数创建组件
```js
const App = () => {
	return <h1>Hello, World</h1>
}
```


如果函数只返回JSX,可以省略`return` 用小括号包裹
```js
const App = () => (
	<h1>Hello, World</h1>
)
```

## 返回多个元素
### 错误示例
```js
function App() {
	return (
		<p>Hello</p>
		<p>World</p>
	)
}
```
这会导致报错

### 正确做法
必须用一个父元素包裹
```js
function App() {
	return (
		<div>
			<p>Hello</p>
			<p>World</p>
		</div>
	)
}
```


## 使用JavaScript变量
React组件中可以直接使用JavaScript
```js
function App() {
  const star = 'Star';
  const planet = 'Planet';

  return (
    <div>
      <h2>Celestial Bodies</h2>

      <p>Name: Polaris, Type: {star}</p>
      <p>Name: Saturn, Type: {planet}</p>
      <p>Name: Sirius, Type: {star}</p>
      <p>Name: Jupiter, Type: {planet}</p>
      <p>Name: Betelgeuse, Type: {star}</p>
      <p>Name: Uranus, Type: {planet}</p>
    </div>
  )
}
```
变量必须写在 `return` 之前，并通过 `{}` 插入到 JSX 中。

最终在页面中，它们会以普通字符串的形式显示。

## 导出组件
组件是可复用的，所以必须导出

```js
function App() {
	return <h1>Hello, World</h1>
}

export default App;
```

## 使用组件

```js
import { createRoot } from 'react-dom/client';
import App from './App';

const root = createRoot(document.getElementById('root'));
root.render(<App />);
```

组件可以像 HTML 标签一样使用。

你也可以多次使用同一个组件：
```js
root.render(
  <div>
    <App />
    <App />
    <App />
  </div>
);
```

## 类组件 vs 函数组件

主要区别在于 **语法**：

- **函数组件**：普通 JavaScript 函数
- **类组件**：继承 `React.Component`，并包含 `render()` 方法

示例：
```js
class App extends React.Component {
  render() {
    return <h1>Hello, World</h1>
  }
}
```

目前你只需要知道：

- 函数组件是后来引入的
- 更简洁、更流行
- 更推荐使用
