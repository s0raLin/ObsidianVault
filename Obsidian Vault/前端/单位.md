CSS使用多种计量单位指定元素的大小
## 绝对单位
具有固定的大小，通常用于处理文本，当你希望某个元素的大小不随环境变化时
### 像素(Pixels)
`px`表示以像素为单位进行测量
```css
p {
	font-size: 16px;
	margin: 10px;
}
```

#### 精确性
对元素外观有像素级的精确控制
#### 可访问性
固定的像素大小在不同屏幕尺寸下适应性较差

#### 整数和小数值
值可以是整数，也可以是小数
```css
p {
	font-size: 12.5px;
	height: 150.5px;
}
```
#### 使用建议
* 适合用于固定布局、图标和图片，这些元素通常需要在所有设备上保持一致
* 建议对文本使用相对单位(如em或rem),这样用户可以根据需要调整文字大小


## 其他绝对单位
### 厘米(cm)
`cm`代表厘米(1cm约38px)
```css
p {
	font-size: 10cm;
}
```
### 毫米(mm)
`mm`代表毫米(1mm=1cm)

```css
p {
	font-size: 100mm; /* 或 10cm */
}
```
### 英寸(in)
`in`代表英寸(1in=2.54cm)

```css
p {
	font-size: 4in; /* 或 10.16cm */
}
```

### 点(pt)
`pt`表示点(1pt=1/72in)

```css
p {
	font-size: 288pt; /* 或 10.16cm */
}
```

### 派卡(pc)
`pc`代表派卡(1pc=12pt)
```css
p {
	font-size: 24pc; /* 或 10.16cm */
}
```

## 相对单位

### 百分比(%)
表示相对于父元素的百分比，在CSS中的百分比是基于父元素相应属性的值来计算的
如果某个 HTML 元素的宽度被设置为 `X%`，那么它的实际宽度可以通过下面的公式计算：`(X / 100) × 父元素的宽度`
例如
```html
<div>
	<p>Units of measure</p>
</div>
```

```css
div {
	height: 400px;
}

p {
	height: 50%; /* 200px */
}
```

### em

相对于父元素字体大小，例如如果字体大小是4.5em，则是父元素字体大小的4.5倍
例如:
```html
<p><b>Units</b> of measure</p>
```

```css
p {
	font-size: 10px;
}

b {
	font-size: 2em; /* 20px */
}
```

### rem
与`em`相似，但有关键区别
始终相对于根元素`<html>`的字体大小
例如

```css
html {
	font-size: 10px;
}

p {
	font-size: 15px;
}

b {
	font-size: 2rem; /* 20px */
}
```


### 默认字体大小

如果你没有显式设置 `<html>` 元素的字体大小，浏览器默认会将其设置为 **16px**。  
在这种情况下：
- `16px = 100%`
- `16px = 1em`
- `16px = 1rem`

你可以基于这个默认值进行计算。


## 视口单位
视口(viewport)指的是用户在不滚动页面情况下看到的网页区域
,主要用于不同设备屏幕的适配

### vw和vh
* 100vw：视口宽度的100%
* 100vh：视口高度的100%

> `100vw` 和 `100vh` **包含滚动条的宽度/高度**，因此在某些情况下会导致出现水平或垂直滚动条。
![[vw和vh.png]]
### vmin
`vmin` 表示视口中**较小尺寸的 1%**。
- 如果视口高度小于宽度：`1vmin = 1vh`
- 如果视口宽度小于高度：`1vmin = 1vw` 
![[vmin.png]]
`100vmin` 表示相对于视口**较小一边的 100%**。

### vmax
`vmax` 表示视口中**较大尺寸的 1%**，与 `vmin` 正好相反。
- 如果宽度大于高度：`1vmax = 1vw`
- 如果高度大于宽度：`1vmax = 1vh`

![[vmax.png]]
`100vmax` 表示相对于视口**较大一边的 100%**。


示例

```html
<body> 
	<div class="minmax">This is an example for vmin and vmax</div> </body>
```

```css
.minmax {
  text-align: center;
  height: 80vmax;
  width: 60vmin;
  background-color: lightcoral;
  font-size: 2rem;
}
```

## 不常用
### ex
用于设置相对于父元素当前字体的x-height(小写字母宽度)的大小
由于不同字体的 x-height 差异较大，这个单位**很少使用**。

### ch
`ch` 表示相对于父元素当前字体中字符 `"0"` 的宽度。
这个单位在**排版和印刷**中非常有用。例如在杂志排版中，可能需要限制一行文本的字符数量。
- 在等宽字体（monospace）中：`1ch ≈ 1 个字符宽度`
- 在比例字体中：字符宽度不一致，因此 `ch` 仅以 `"0"` 的宽度为基准

示例：

```css
p {
  font-size: 5ch;
}
```


## 单位值的书写规则
- 单位值可以是整数：`5vh`、`27%`、`5vw`
- 也可以是小数：`5.6em`、`1.2rem`、`25.5%`

### 特殊规则
- 如果值为 `0`，单位可以省略
    `font-size: 0;`
- 小数可以省略前导 `0`
    `font-size: .9em;`
