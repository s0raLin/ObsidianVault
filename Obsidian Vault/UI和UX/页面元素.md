## 页面元素的类型
### 输入控件
允许用户输入或提交信息的元素
### 导航组件
显示用户在应用结构的位置，在不同页面中移动
### 信息组件
显示状态或提示信息的元素
### 容器
按照不同属性对信息进行分组的元素

## 常见的UI组件库
开发者不需要从零开始“造轮子”，只需在标准组件基础上进行样式调整即可。
目前最常见的两个 UI 组件库是：

### Android Material Design（Material You）
- [https://material.io](https://material.io)
- [https://materialdesignkit.com/android-gui/](https://materialdesignkit.com/android-gui/)

### iOS UI Kit
- [https://developer.apple.com/design/resources/](https://developer.apple.com/design/resources/)
