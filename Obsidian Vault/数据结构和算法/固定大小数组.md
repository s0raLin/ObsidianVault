一种在创建时固定大小的、用于存储相同类型集合的结构


## 示例

### 浮点数数组
```
measurements = [10.8, 14.3, 13.5, 12.1, 9.7]
```
![[浮点数数组.png]]

### 字符串数组

```
flowers = ["Rose", "Tulip", "Lily", "Orchid", "Sunflower"]
```
![[字符数组.png]]

* 数组的**大小(size)** 表示数组中包含多少个元素，也称为**长度(length)**
* 每个元素的**索引(index)** 是一个数字，大多数编程语言的**索引(index)** 从0开始

## 定义数组
```python
measurements = [10.8, 14.3, 13.5, 12.1, 9.7]
```

## 访问元素
编程语言可能提供了多种方法，最常用的是**通过索引访问元素**

例如，访问`measurements`数组的第三个元素(索引为2)
```python
value = measurements[2] # value = 13.5
```
![[访问元素.png]]
## 修改元素
**通过索引修改元素**
```python
measurements[2] = 3.7
```
修改元素后的数组如下
![[修改元素.svg]]
读取和修改操作的时间复杂度都是 **O(1)**

## 插入和删除元素
简而言之，**在固定大小数组中插入或删除元素是不可能的**,因为这些操作会改变数组的长度，而固定大小数组的长度是不可变的。

![[插入和删除元素.png]]

这种操作需要创建一个新的数组，并复制原有元素，同时加入新元素，因此时间复杂度为 **O(n)**


### 示例
计算数组均值

```python
# 0. 定义数组
measurements = [10.8, 14.3, 13.5, 12.1, 9.7]

# 1. 初始化变量sum = 0
sum = 0

# 2. 遍历数组并将每个元素加到sum上
for i in (1, len(measurements)):
    sum = sum + measurements[i]

# 3. 计算均值
mean = sum / len(measurements) # 12.08
```
这里，`len(measurements)` 表示数组长度
