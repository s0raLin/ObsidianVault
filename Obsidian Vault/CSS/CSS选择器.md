CSS选择器可以选择特定的HTML元素并设置样式
## 基本选择器
### 元素选择器
```css
p {
	color: brown;
}
```
### id选择器
```css
#big {
	font-size: 30px;
}
```
### class选择器
```css
.blue {
	background-color: blue;
}
```

### 通用CSS选择器
选择HTML文档中的所有元素
```css
* {
	/* 设置一致的基本样式*/  
}
```

### 特异性
特异性是一种度量，用于确定当多个规则匹配特定元素时应用哪个CSS规则。这基于CSS规则中使用的不同类型选择器来计算。从最高到最低的特异性层次是内联样式、ID、类、属性和元素。例如，ID选择器(#example)的特异性高于类选择器(.example)，后者又高于元素选择器(div)。特异性是基于每个选择器计算的，在两个选择器具有相同特异性时，将应用CSS中定义的最新规则。

### 继承
在CSS中，继承控制属性如何从父元素流向其子元素，这有助于避免属性重复，并使样式表更容易管理。然而，并非所有CSS属性都是可继承的。与文本格式相关的属性，如font-family、color和font-size，通常由子元素继承，但与盒模型相关的属性如width、height、margin和border则不被继承。这意味着除非明确指定，否则子元素不会从其父元素继承盒模型属性。
